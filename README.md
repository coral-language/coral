## About

Coral combines the best of both worlds: Python's elegant syntax and Rust's memory safety guarantees. Built from the ground up in Rust, Coral offers:

- **üêç Python-like Syntax**: Familiar, readable code with modern language features
- **ü¶Ä Rust Memory Safety**: Zero-cost abstractions with compile-time memory safety
- **‚ö° High Performance**: Fast parsing, compilation, and execution
